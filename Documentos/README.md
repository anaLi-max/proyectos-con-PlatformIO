# Proyecto ESP32 con DHT y ThingSpeak

## Descripción
Este proyecto consiste en la configuración y programación de una ESP32 utilizando PlatformIO en Visual Studio Code.

Se configuraron las librerías:
- DHT (11 o 22) para medición de temperatura y humedad
- ThingSpeak para el envío de datos a la nube

## Herramientas utilizadas
- ESP32
- Sensor DHT11 / DHT22
- PlatformIO
- Visual Studio Code
- GitHub
